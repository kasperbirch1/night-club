# REST API

Dette API indeholder alle data til Night Club.

## Installation
```
npm install
```

## Kør
```
npm run dev
```
eller
```
npm start
```

## Reset
Du skal kører /reset endpointet for at fylde data i. </br>
Hvis du har brug for at resette data, kan du ligeledes kører /reset endpointet
```
http://localhost:4000/reset
```
